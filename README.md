# Java Simple Projects

This repository contains small Java programs and exercises that I build while learning the Java programming language.

The goal of this repository is to practice core Java concepts through simple examples and mini-projects.

## Topics Covered

Examples in this repository may include:

- Variables and data types  
- Control flow (if, switch, loops)  
- Methods and parameters  
- Arrays and collections  
- Object-Oriented Programming in Java (classes, objects, inheritance, polymorphism)  
- Basic problem solving  
- Small console-based programs  

## Purpose

This repository serves as:

- A personal learning journal for Java  
- A collection of simple practice programs  
- A reference for basic Java concepts  

## Future Plans

As I progress, I plan to add:

- More advanced Java examples  
- Algorithm and data structure practice  
- Object-Oriented mini‑projects  
- Java-based console applications
